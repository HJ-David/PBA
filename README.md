Company website for **Page Backcountry Adventures**, built with [Next.js](https://nextjs.org).

The homepage hero uses a Cloudinary video (with a local fallback at `public/videos/intro.mp4`).

## Getting Started

From the repo root, install deps and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

### Node version

This project is intended to run on **Node.js 22+**. If you use `nvm`, a `.nvmrc` is included.
